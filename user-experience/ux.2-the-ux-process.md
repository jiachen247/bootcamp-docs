# UX.2: The UX Process

User experience design applies design thinking to the creation of digital products.

{% embed url="https://youtu.be/K44tP606seE" %}

Steps in the UX design process:

1. User Research
2. User journey maps
   1. Customer Journey Maps
   2. Service Blueprints
   3. Experience Maps
   4. Spatial Maps
   5. Mental Model Diagram
3. Design
4.  Wireframe
5. Prototype
   1. Low-fidelity
   2. High-fidelity
6. User testing
   1. Moderated
   2. Remote Moderated
   3. Unmoderated
   4. a/b testing
7. UI Design
8. Development
9. Launch
10. Iteration
