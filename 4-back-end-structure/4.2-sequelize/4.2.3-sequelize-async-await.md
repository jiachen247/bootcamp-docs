# 4.2.3: Sequelize Async Await

## Query Related Tables Using Async

```javascript
import db from './models/index.mjs';

const getData = async () => {

  try{
    const category = await db.Category.findOne({
      where: {
        name: [process.argv[2]]
      }
    });
    
    const categoryItems = await category.getItems();
    
    console.log( categoryItems );
  }catch( error ){
  
    console.log( error );
  }
};

getData();
```
